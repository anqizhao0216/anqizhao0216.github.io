[LeetCode]29.Divide Two Integers

Posted on 2017-11-12

Description: Divide two integers without using multiplication, division and mod
operator.
If it is overflow, return MAX_INT.

近似除法。
不能用乘除，就只能用加减，以及位运算来操作了。
可以从被除数当中挨个减去除数，直到被除数小于0.那么减掉的次数也就是要求的近似商。

当除数非常小的时候，很显然这个方法用时将会很久。

可以用二分法来加速这个过程。“不断对除数*2， 直到它恰好比被除数小（即再乘2就会比被除数大）为止。
加倍的同时，也记录下cnt, 将被除数减掉加倍后的除数，并且结果+cnt。因为是2倍地加大，所以速度会很
快，指数级的速度。”

乘2的原因是因为用位运算的左移操作可以代替，如果选择别的数值，无法使用位运算？

这道题的难点不在于想到用位运算来代替乘2，而在于对各种Corner cases的处理。

  • 除数为0。 被除数为正，结果为正无穷；否则为负无穷
  • 被除数为0。 直接返回0
  • 被除数，除数小于0。
  • 整型溢出的问题 记录结果的符号，同号为正，异号为负。然后取绝对值进行运算

    注意： 因为int最大2147483647，最小-2147483648，最小值直接abs后会溢出，所以要先将被除数与
    除数转换为long之后再取绝对值，以防出现溢出的情况。

每次将除数左移一定的位数，比如i位，使得除数恰好小于被除数，此时相当于b2^i,然后再对剩下的数(a-
b2^i)做刚才的操作，直到剩下的数小于b为止。看b总共乘以了多少power of 2, 将其加和即可。
